Discord Bot

Developed a Discord bot to provide members of the Left 4 Dead Underworld Co-op steam group with real-time access to server information. Utilized a custom Python script to automate the creation of JS files which links to Roshal Archive Compressed or Valve Pak files containing custom maps stored on Google Drive. Additionally, implemented custom commands that allow users to retrieve server information in real-time such as the number of slots, IP address, and available server commands.

L4DAutoConfig

Designed and developed a video game modification add-on which enhances the in-game UI, changes the in-game font, and revamps the main menu design, created for members of a video game group by utilizing Notepad++ to create Valve Resource Files. Improved the gameplay experience by simplifying access to server commands, providing the best settings for users with low-end hardware.
